## ✨ 주요 기능

- **📅 블록 일정** — 26개 × 14일 블록별 일자별 상세 일정 (펼치기/접기)
- **🗺️ 코스 목록** — 278개 전체 코스 테이블 (거리·시간·도시·국가유산)
- **🏨 숙소 가이드** — 56개 거점도시 추천 호텔/리조트
- **💰 비용 계산기** — 인원·숙박비 설정으로 블록별/전체 예산 산출
- **⏰ 일과 시간표** — 트레킹일/이동일/자유일정일 표준 스케줄
- **🏛️ 국가유산 10길** — 트레일 연계 현황
- **📊 총괄 일정표** — 전체 26블록 한눈에 보기
- **🔍 검색** — 코스명·도시·숙소·국가유산 실시간 검색
- **🎛️ 트레일 필터** — 해파랑길/남파랑길/서해랑길/제주올레길 개별 조회

## 🚀 GitHub Pages 배포 방법

